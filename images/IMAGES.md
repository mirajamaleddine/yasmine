# Illustration drop-in guide

The site currently uses **placeholder images**. Replace each file below with the
real illustration, keeping the **exact same filename** — no code changes needed.

| Filename | What goes here |
|---|---|
| `monogram.png` | The "R & Y" monogram |
| `couple.png` | The bride & groom in the garden (hero) |
| `venue-house.png` | The little house / venue illustration |
| `map.png` | The hand-drawn map of Ashrafieh |
| `envelope-heart.png` | Envelope with the red heart |
| `cake.png` | The three-tier wedding cake |
| `cat-winged.png` | The winged (pegasus) cat — floats across the screen |
| `cat-bowtie.png` | The orange bow-tie cat — trots along the bottom |
| `hedge-border.png` | The green hedge strip (footer) |
| `garden-bg.png` | Optional spare garden/pond art (not yet placed) |

**Tips**
- Keep them as **PNG with transparent backgrounds** (yours already are).
- Filenames are case-sensitive on Netlify — match exactly, all lowercase.
- No need to resize; the CSS scales them. But smaller files load faster —
  if any are huge (>1 MB), shrink them at https://squoosh.app first.
